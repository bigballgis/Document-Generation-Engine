# Product Documentation

This folder contains product-facing requirements.

## Files

- [PRD.md](PRD.md): product requirements, scope, user capabilities, lifecycle behavior, API behavior, audit, and open product questions.
- [usability-review.md](usability-review.md): usability, ease-of-use, and friendliness review backlog.
- [authoring-rendering-first-principles-review.md](authoring-rendering-first-principles-review.md): first-principles review for template authoring, controlled rich text, DOCX fidelity, reusable content, and product weight risks.

## Update Rule

Update this folder when product behavior, user capability, lifecycle, API behavior, output behavior, or user-facing scope changes.
