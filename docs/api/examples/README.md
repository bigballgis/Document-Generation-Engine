# API Examples

These examples accompany [openapi-v1.yaml](../openapi-v1.yaml).

Rules:

- Do not put real API credential secrets, customer data, account numbers, identity numbers, amounts, encryption passwords, complete download URLs, or complete AD Group membership in examples.
- Resource IDs use opaque examples such as `TASK-7K3M9Q2R`, `BATCH-7K3M9Q2R`, and `DOC-8F2N6P4Q`.
- Download URLs are illustrative and must be masked in logs, audit records, management views, and documentation examples.
