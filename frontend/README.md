# Frontend

Vue 3 + TypeScript + Vite + Element Plus + Pinia + Vue Router + vue-i18n (English base).

## Commands

```powershell
pnpm install
pnpm dev
pnpm lint
pnpm type-check
pnpm test
pnpm build
```

## P0 scope

- Login route shell (no backend auth wiring yet — P1)
- REDBC / GREENBC theme tokens via CSS variables
- English-first i18n message keys
